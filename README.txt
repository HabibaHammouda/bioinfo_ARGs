This folder contains all required files for the work of Team 9 in the Final Project in Bioinformatics I course.

files required to successfully run this application are:
1) ARGs_final_script.R
2) SRR797943.fastq
3) nucleotide_fasta_protein_homolog_model.FASTA
4) aro_categories_index.tsv
The rest of the files are output from running the R script

Files Description:
- ARGs_final_script.R - This is the R script.
- Report (pdf and word)
- SRR797943.fastq - raw fastq data downloaded using SRA toolkit
- SRR797943.fasta - the fastq is converted to fasta file
- nucleotide_fasta_protein_homolog_model.FASTA - the CARD database (used for comparison in BLAST)
- results_SRR797943 - BLAST results
- aro_categories_index.tsv - file used in annotation (to map the IDs with the Gene Families)
- metagenomics ARGs research paper - Reference Paper
